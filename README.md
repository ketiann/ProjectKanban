# 项目数据可视化看板

数智技术服务中心项目数据可视化大屏，基于源数据Excel文件一键生成HTML看板。

## 功能特性

- **一键生成**：运行 `python generate_dashboard.py` 即可自动生成看板
- **自动发现数据**：脚本自动识别同目录下的4个源数据文件
- **22个数据维度**：覆盖项目、合同、人力、进度、预警等
- **数据溯源**：每个看板数据均可溯源至源数据文件的具体Sheet和字段
- **深色科技风**：ECharts 5.5.0 图表渲染，响应式布局

## 快速开始

### 1. 环境准备

```bash
# 需要 Python 3.7+
pip install pandas openpyxl
```

### 2. 准备源数据

将以下4个文件放到同一目录：

- `项目数据表.xlsx`
- `人力投入表.xlsx`
- `本期人力投入项目情况(YYYY-MM-DD开始截至YYYY-MM-DD).xlsx`
- `本期项目变化情况(YYYY-MM-DD开始截至YYYY-MM-DD).xlsx`

### 3. 生成看板

```bash
python generate_dashboard.py
```

执行后同目录下生成 `看板.html`，用浏览器打开即可查看。

### 4. 数据更新

替换源数据文件（保持格式一致），重新运行脚本即可生成新看板。

## 看板模块清单

1. KPI指标卡（项目总数、合同金额、回款、成本）
2. 本月人力投入情况
3. 合同类型分布 / 项目类型分布
4. 回款率分布 / 项目进度分布
5. 本月部门人力投入（数智技术服务中心）
6. 人员工作饱和度分析
7. 部门工时投入对比
8. 部门人均工作饱和度
9. 本期人力投入TOP10项目（按部门着色）
10. 各部门人力投入TOP3项目
11. 本期项目变化动态
12. 项目超期/到期预警
13. 合同金额TOP10 / 人力投入累计TOP10
14. 本期人力投入项目明细
15. 挂起项目 / 低饱和度人员

## 技术栈

- **数据处理**：Python + pandas + openpyxl
- **图表渲染**：ECharts 5.5.0
- **页面布局**：原生 HTML / CSS / JavaScript
- **主题**：深色科技风（#0a0e27 背景）
